# Dossier Images

Ce dossier contient les images du site web de Mecquignies.

## Logo de la commune

Pour ajouter le logo officiel de Mecquignies :

1. **Télécharger le logo depuis le site officiel** :
   - Visitez [https://mecquignies.fr/](https://mecquignies.fr/)
   - Faites un clic droit sur le logo en haut de la page
   - Sélectionnez "Enregistrer l'image sous..."
   - Nommez le fichier `logo-mecquignies.png`

2. **Ou contacter la mairie** :
   - Email : mairie@mecquignies.fr
   - Téléphone : 03 27 63 17 09
   - Demandez une version haute résolution du logo officiel

3. **Placer le fichier** :
   - Copiez le fichier `logo-mecquignies.png` dans ce dossier
   - Le logo apparaîtra automatiquement dans l'en-tête du site

## Formats recommandés

- **Format** : PNG avec fond transparent (recommandé) ou JPG
- **Dimensions** : Hauteur de 60-80px (la largeur s'adaptera automatiquement)
- **Poids** : Maximum 200 KB pour optimiser le chargement

## Autres images

Vous pouvez ajouter d'autres images dans ce dossier :
- Photos du village
- Photos du patrimoine (église, château)
- Photos d'événements
- etc.

Nommez vos fichiers de manière descriptive :
